Full Adder RTL to GDSII using OpenLane

Project Overview

This project demonstrates the complete RTL-to-GDSII ASIC design flow for a 1-bit Full Adder using OpenLane.

Tools Used

- Verilog HDL
- OpenLane
- OpenROAD
- Magic VLSI
- KLayout
- Ubuntu Linux

Project Files

- "src/" – Verilog source files
- "configuration" – OpenLane configuration
- "gds/" – Final GDSII layout
- "screenshots/" – Floorplan, Placement, CTS, Routing and Final GDS screenshots

ASIC Design Flow

1. RTL Design
2. Functional Simulation
3. Logic Synthesis
4. Floorplanning
5. Placement
6. Clock Tree Synthesis (CTS)
7. Routing
8. GDSII Generation

Output

- Final GDSII Layout ("FA.gds")
- Verilog Source Code
- OpenLane Configuration
- Physical Design Screenshots
